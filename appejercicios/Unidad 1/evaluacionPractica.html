Formatos

Titulos

HTML dispone de un elemento < Hx >< /Hx > que se pueden utilizar para definir que partes de nuestro texto deben ser consideradas como encabezados (de seccion, capitulo, etc). La etiqueta asigna un mayor tamaño de texto a los caracteres (en funcion de x, donde x es un numero entre el 1 y el 6). Ademas, se utiliza un tipo de letra en negrilla para la cabecera e incluye un salto de parrafo despues de esta cabecera

Formatos

HTML proporciona una etiqueta para gestionar la tipografia. Esta etiqueta, FONT, ha sido declarada obsoleta en HTML 4.01, por lo que se debe evitar su uso y tratar de usar en su lugar hojas de estilo (CSS).
Al utilizar esta etiqueta, se debe de hacer con precaucion al especificar la tipografia ya que se debe tener en cuenta que quiza el cliente no tenga dicha tipografia instalada, con lo que la vision que obtendra de la pagina no corresponda a la deseada. Los atributos soportados por FONT y que permiten definir las caracteristicas del tipo de letra son:

SIZE (Indica el tamaño de los caracteres, con valores de 1 a 7)
COLOR (color de los caracteres)
FACE (Tipografia a usar, se pueden especificar mas de una, separadas por comas)
Para esto, necesitamos utilizar la etiqueta < font>< /font> Un ejemplo de esto es: < font SIZE=1>1< /font>

Formatos Adicionales

Ademas de los formatos vistos anteriormente, HTML dispone tambien de algunos tipos logicos de letra, que los navegadores pueden representar de otra forma:

EM (Enfatizado) < em >< /em>
STRONG (Destacado) < STRONG>< /STRONG>
CODE (Codigo de programa) < code>< /code>
CITE (Cita) < cite>< /cite>

Estilos de Letras

HTML proporciona un conjunto de etiquetas que permiten definir diferentes estilos de letras para el texto contenido entre las siguientes marcas:
< b>< /b> (Negrita)                            < i>< /i> (Italica)
< u>< /u> (Subrayado)                      < strike>< /strike> (Tachado)
< sup>< /sup> (Superindice)             < sub>< /sub> (SubIndice)
< blink>< /blink> (Parpadeo)            < tt>< /tt> (Teletipo)
< big>< /big> (Grande) (Parpadeo)  < small>< /small>(Pequeño)

Colores

Tambien podemos agregar colores a nuestros textos con la propiedad color seguido de un numero en exadecimal.
Un ejemplo de esta propiedad es: < font color="#800000">C< /font>

Parrafos

Para los parrafos, se utiliza la etiqueta < p>, HTML ignora los saltos de linea introducidos en el fichero original, por lo que para HTML todo el texto es continuo, por lo que un mecanismo que se utiliza para indicar principio y final de parrafo, esta esta indicando con las etiquetas < p> y < /p>

Parrafos Citados

HTML dispone de un elemento BLOCKQUOTE, que permite representar parrafos citados literalmente de otro texto. Generalmente se representan identando a derecha e izquierda y con un salto de parrafo antes y despues del parrafo citado.
Se deberia evitar el uso de BLOCKQUOTE para identar texto, puesto que el navegador puede representarlo de otra forma, por ejemplo, no identado.
Ejemplo de la etiqueta BLOCKQUOTE: < blockquote>< /blockquote>

Divisiones

El elemento < div>< /div> permite dividir el texto en bloques, insertando entre los bloques un salto de linea simple, al igual que < br>, pero, a diferencia de este, admite los mismo atributos que P, es decir, se puede definir la alineacion del texto para cada bloque DIV

Las alineaciones soportadas por DIV, son establecidas mediante el parametro ALIGN, estas son:
LEFT, alineacion a la izquierda, es el activo por defecto
RIGHT, alineacion a la derecha
CENTER, centrado del texto.

Listas

Las listas en HTML nos permite crear conjuntos de elementos en forma de lista dentro de una página, todos los cuales irán precedidos, generalmente, por un guión o número.

Listas Ordenadas

Las listas en HTML ordenadas son aquellas que nos muestran los elementos de la lista en orden. Para representar el orden tendremos los elementos numerados. Es decir, cada uno de los elementos irá precedido de un número o letra que establezca su orden.
Las listas en HTML ordenadas se representan mediante el elemento OL.
Cada uno de los elementos de la lista ordenada se representará mediante el elemento LI.
< ol>
< li>Julio< /li>
< li>Carmen< /li>
< li>Ignacio< /li>
< li>Elena< /li>
< /ol>

Listas No Ordenadas

Las listas desordenadas en HTML nos sirven para mostrar los elementos sin ningún tipo de orden, simplemente precedidos por una viñeta que puede ser un punto, un cuadrado
Para representar los elementos de la lista desordenada utilizamos el mismo elemento que con las listas ordenadas, es decir, el elemento li.
< ul>
< li>Elemento 1< /li>
< li>Elemento 2< /li>
...
< li>Elemento N< /li>
< /ul>

Enlaces

Lo más importante de los documentos HTML son los enlaces. Ya que mediante los enlaces en HTML podemos comunicar una página con otra. De esta forma, enlazando documentos HTML podemos acabar tejiendo lo que es Internet. Para crear un enlace en HTML utilizamos el elemento A con la siguiente sintaxis: < a>Contenido del enlace< /a>

Pero solo con esto el enlace no tendrá mucha utilidad ya que el principal objetivo del enlace es enlazar a un destino. Para poder indicar el destino de un enlace utilizamos el atributo href. En valor del atributo href puede ser cualquier URI que represente un recurso. Es decir, una página, una parte de una página, una URL genérica, un archivo,… De esta forma el enlace en HTML lo crearemos con la sintaxis: < a href="URI">Contenido del enlace< /a>

Otra capacidad que tenemos en HTML es la de enlazar a una parte concreta del documento. Imagina que en un documento tenemos un título y varios capítulos. Y lo que queremos hacer desde otro documento HTML o bien desde el mismo documento es enlazar directamente al inicio de un capítulo.

La sintaxis sera: < a name="partedocumento">Contenido< /a>

Imagenes

Para poder incluir imagenes y graficos en una paginam, utilizamos la etiqueta < img src="" alt="">
IMG dispone de diversos atributos que nos permiten especificar que fichero de imagen se quiere usar, las medidas de esta, etc. El atributo que permite especificar que imagen se va a mostrar es SRC

Otros atributos importantes a usar en la etiqueta < img> y que permiten especificar las medidas de la imagen (ancho y alto), width y height. En caso de no especificar los atributos, el navegador mostrara la imagen con el tamaño que tenga el archivo original.
Un ejemplo de la etiqueta es: < img src="img/logo-toyota.png" width="30%" height="30%">

Tablas

Las tablas es el elemento del lenguaje HTML que utilizaremos para mostrar información de forma agrupada. Ya sea texto, imágenes, vídeos,…
El elemento table será el que nos ayudará para crear las tablas en HTML.
Un mal uso de las tablas en HTML es el motivo de maquetación, uso que se dió a las tablas en los principios del lenguaje HTML. Algo que hoy en día es una mala práctica. Cambiando por un modelo de maquetación apoyado en capas. Para crear una tabla vamos a necesitar conocer, al menos, tres elementos: table, tr y td. Si bien existen otros cuantos que nos permitirán ampliar la funcionalidad de las tablas.
El primer elemento es table. table es el elemento que representa las tablas y será el que agrupe al resto de elementos. Por lo tanto tiene sus etiquetas de inicio y cierre. Siguiendo con la construcción de la tabla el siguiente elemento que necesitamos es tr. El elemento tr representa a una fila de la tabla. Por lo tanto tendremos tantos elementos tr como filas tenga la tabla.
Así, si queremos tener una tabla de tres filas tendremos un código como el que sigue:
< table>
< tr> ... < /tr>
< tr> ... < /tr>
< tr> ... < /tr>
< /table>
Una tabla de tres filas por cuatro columnas quedaría de la siguiente forma:
< table>
< tr>
     < td>Fila 1 Columna 1< /td>
     < td>Fila 1 Columna 2< /td>
     < td>Fila 1 Columna 3< /td>
     < td>Fila 1 Columna 4< /td>
< /tr>
< tr>
     < td>Fila 2 Columna 1< /td>
     < td>Fila 2 Columna 2< /td>
     < td>Fila 2 Columna 3< /td>
     < td>Fila 2 Columna 4< /td>
< /tr>
< tr>
     < td>Fila 3 Columna 1< /td>
     < td>Fila 3 Columna 2< /td>
     < td>Fila 3 Columna 3< /td>
     < td>Fila 3 Columna 4< /td>
< /tr>
< /table>
